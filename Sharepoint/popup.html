Search List

Bulk search for values in a specific SharePoint field

📖 Read More - User Guide

How to Use Search List

  1. Select a List: Click "Get All Lists" to fetch available lists, then select the list you want to search.
  2. Choose a Column: Click "Get Columns" to see all available columns, then select the field to search in.
  3. Select Comparison Operator:
    • Equals (=): Finds items where the field exactly matches the search value (exact match).
    • Contains: Finds items where the field includes the search value anywhere (partial match).
  4. Enter Search Values: You can enter multiple values in two ways:
    • Comma-separated: Value1, Value2, Value3
    • Multiline: Put each value on a new line
    • Mixed: You can use both commas and newlines together
    The search will find items matching ANY of the values (OR logic).
  5. Set Row Limit: Optional - limit results to prevent large datasets. Set to 0 for unlimited.
  6. Click Search: View results in the table below.

Examples

Example 1: Search for specific statuses

Column: Status
Operator: Equals (=)
Values: Completed, In Progress, Pending

Result: Finds items where Status is exactly "Completed" OR "In Progress" OR "Pending"

Example 2: Search for partial matches

Column: Title
Operator: Contains
Values:
Project
Task
Report

Result: Finds items where Title contains "Project" OR "Task" OR "Report" anywhere in the text

Tips

  • Use Contains for flexible searches when you're not sure of exact values.
  • Use Equals (=) for precise searches when you need exact matches.
  • Values are case-insensitive in SharePoint searches.
  • Click "Export Results to CSV" to save your findings for further analysis.
  • Clear the form to start a new search with different parameters.
"Equals" matches the exact value. "Contains" matches if the field includes the value anywhere.
