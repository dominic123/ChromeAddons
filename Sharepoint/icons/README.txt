# Icons Required

Place the following PNG files in this folder:

- icon16.png  - 16x16 pixels
- icon32.png  - 32x32 pixels
- icon48.png  - 48x48 pixels
- icon128.png - 128x128 pixels

You can use any icon generator tool or create simple square PNG images with a SharePoint-like design.

For testing, you can create simple colored square images with any image editor.

Quick option: Use https://www.favicon-generator.org/ or similar tools to generate icons from an image.
